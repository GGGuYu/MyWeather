## ✨ 功能特点

### 🎯 核心功能
- **📍 实时定位** - 自动获取当前位置的天气信息
- **🌡️ 精准预报** - 实时温度、湿度、风速、空气质量指数(AQI)
- **📅 未来天气** - 未来几天的天气预报趋势
- **⏰ 小时预报** - 每3小时详细天气预报

### 🤖 AI 智能助手
- **💬 天气总结** - 一键询问 AI，获取今日天气总结
- **👔 穿搭推荐** - 根据天气智能推荐穿衣搭配
- **🎒 出行建议** - 雨伞、外套、防晒霜... AI 贴心提醒

### 🎨 界面设计
- 采用 Material Design 3 设计语言
- 流畅的动画过渡效果
- 清晰直观的天气图标展示

## 🚀 快速开始

### 环境要求
- Android Studio 最新版
- JDK 11 或更高
- Android SDK API 26+

### 运行步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/GGGuYu/MyWeather.git
   cd MyWeather
   ```

2. **打开项目**
   - 使用 Android Studio 打开项目
   - 等待 Gradle 同步完成

3. **配置 API Key** ⚠️
   - 在 `app/src/main/java/com/ixuea/course/weather/config/Config.kt` 中配置天气 API Key

4. **运行应用**
   - 连接设备或启动模拟器
   - 点击运行按钮 ▶️
